| 策略指导 | 限制规则 |

| 解释"为什么" | 只说"不能做" |
| 提供"怎么办" | 不说"怎么做" |
| 适用于所有场景 | 只针对特定场景 |
| 举一反三 | 死记硬背 |
| 理解核心原则 | 记住具体规则 |

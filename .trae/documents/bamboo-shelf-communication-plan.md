# 竹子植物架 - 需求沟通方案

## 当前问题
经过多轮调整，竹子架的效果仍然不符合您的预期。这说明我们在**视觉理解**上存在偏差。

## 推荐方案（按优先级排序）

### 方案 1: 提供参考图片 ⭐⭐⭐ 最推荐
**为什么最好**：
- 图片能直接传达"你想要的样子"
- 我可以分析图片中的颜色、形状、风格等细节
- 避免文字描述的歧义

**您可以提供**：
- 从网上找一张类似的竹子架子图片
- 或者手绘一个草图拍照给我

### 方案 2: 详细描述具体特征 ⭐⭐
如果您没有参考图片，可以详细描述：

**关于竹子**：
1. **竹子的形态**：是直的还是弯曲的？是粗还是细？
2. **竹节的样式**：竹节是凸起的环状？还是只是颜色深一点？
3. **颜色**：想要什么色调的绿色？（嫩绿、翠绿、墨绿、青绿）
4. **质感**：光滑的？有纹理的？

**关于藤蔓**：
1. **藤蔓的位置**：缠绕在竹子上？还是在旁边？
2. **藤蔓的样子**：细线状？还是有明显的主茎？
3. **叶子的数量和大小**：每侧几片叶子？大还是小？
4. **整体感觉**：茂盛的？稀疏的？

### 方案 3: 告诉我不想要的 ✅
有时候说清楚"不要什么"也很有用：
- "不要像现在这样..."
- "太复杂了/太简单了"
- "看起来很假"
- "像塑料的"

## 我的建议

**最佳组合**：参考图片 + 简单描述

例如：
> "我想要这种风格的竹子架 [图片]，藤蔓要自然一些，不要太夸张"

# 修复"AI 过度解读 emoji"问题

## 🐛 问题描述

**用户反馈**：
- 用户发了一个🌡️温度计 emoji
- AI 直接解读为"查看天气"
- 输出一大堆天气信息模板：
  ```
  **当前天气信息**
  日期：2024-02-28
  地区：[用户所在地区]
  白天温度：[白天温度]℃
  风力：[风力等级]
  ...
  ```

**问题本质**：
- ❌ AI 过度解读用户意图
- ❌ 看到 emoji 就输出知识模板
- ❌ 不像真人朋友，像机器人答题

## 🎯 真人感的正确做法

当用户发 emoji 时，真人朋友会：

**✅ 正确示范**：
```
用户："🌡️"
AI："哈哈，您发个温度计是想考我什么呀？🤔
     是想问植物的适宜温度？
     还是想问今天的天气？
     或者您花园里的植物需要调温？
     给我点提示呗～😄"
```

**❌ 错误示范**：
```
用户："🌡️"
AI："**当前天气信息**
     日期：2024-02-28
     地区：[用户所在地区]
     白天温度：[白天温度]℃
     ...（输出一堆模板信息）"
```

### 1. 添加【严禁行为】总则

在 SYSTEM_PROMPT 开头添加**最高优先级的禁令**：

```javascript

⚠️ **绝对禁止的行为（违反就不是真人感）**：

1. **看到 emoji 就输出知识模板**
   - ❌ 用户："🌡️" → AI 输出天气信息模板
   - ❌ 用户："💧" → AI 输出浇水知识大全

   - ❌ 用户发个表情，你就当是问题来回答
   - ❌ 用户没说清楚，你就把所有相关信息都说了
   - ✅ 正确：意图不明就反问，不要瞎猜

3. **一本正经地输出模板信息**
   - ❌ 输出带方括号的模板：[用户所在地区]、[白天温度]
   - ❌ 输出格式化的知识列表：1、2、3、4...
   - ✅ 正确：像聊天一样说话，不要用模板

4. **忽视用户的真实需求**
   - ❌ 用户可能只是开玩笑，你却认真回答
   - ❌ 用户可能在测试你，你却输出长篇大论
   - ✅ 正确：先问清楚，再回答
```

## 🎓 教学要点（给初学者）

### 1. 什么是"真人感"？

**真人感的核心**：像朋友聊天，不是机器人答题

| 真人朋友 | 机器人 AI |
|----------|----------|
| 你发个表情，朋友会问"你想说啥？" | 你发个表情，AI 就开始讲课 |
| 你没说清楚，朋友会追问 | 你没说清楚，AI 就瞎猜 |
| 你开玩笑，朋友会跟着笑 | 你开玩笑，AI 一本正经 |
| 朋友说话有温度、有情感 | AI 说话像模板、冷冰冰 |

### 2. 为什么 AI 会过度解读？

**原因**：
1. **训练数据问题** - AI 看过太多"问答对"，习惯了一问一答
2. **缺乏上下文理解** - AI 不知道用户可能只是开玩笑
3. **过度优化** - 想让 AI 显得"专业"，结果变得死板

**解决方法**：
- 明确告诉 AI 什么**不能做**（禁令）
- 提供正确的**示范**（示例）
- 设定严格的**约束**（字数、反问等）

### 3. 如何设计 AI 的人设？

**好的人设设计**：
```javascript
【人设定位】
- 专业但不死板：有知识，但不用术语
- 幽默但有分寸：开玩笑，不过分
- 温暖但不八婆：关心用户，不话多
- 真诚但不呆板：有情感，不做作

【🚫 严禁行为】
- 禁止看到 emoji 就输出知识
- 禁止过度解读用户意图
- 禁止输出模板信息
- 禁止忽视用户真实需求

【✅ 正确示范】
- 提供具体的对话示例
- 展示什么是"对"，什么是"错"
```

**差的人设设计**：
```javascript
// ❌ 太空洞
"你是一个友好的 AI 助手"

// ❌ 太模糊
"回答用户的问题，提供帮助"

// ❌ 没有约束
"可以适当幽默"（但没说什么时候不能幽默）
```

### 4. Prompt Engineering 的核心技巧

**技巧 1：明确禁令**
```javascript
❌ 差："不要输出太多信息"
✅ 好："回复必须 < 80 字，必须包含至少 1 个反问"
```

**技巧 2：提供示例**
```javascript
❌ 差："要像朋友一样聊天"
✅ 好：
用户："🌡️"
AI："哈哈，您发个温度计是想考我什么呀？🤔"
```

**技巧 3：设定优先级**
```javascript
❌ 差：（没有优先级）
✅ 好："最高优先级"、"严禁"、"绝对禁止"
```

**技巧 4：重复强调**
```javascript
// 在多个地方重复同一规则
【🚫 严禁行为】提到
【互动式对话】详细说
【回复长度控制】再强调
```

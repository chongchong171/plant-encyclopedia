## 💡 推荐方案（按需求）

### 🏆 最便宜：中国邮政小包（挂号）
- **费用：** ¥90-120 到美国，¥100-140 到加拿大
- **时效：** 15-40 天
- **适合：** 不紧急、低客单价产品
- **缺点：** 时效慢、追踪信息不完整

### ⚡ 性价比最高：ePacket（国际 E 邮宝）
- **费用：** ¥100-140 到美国，¥120-160 到加拿大
- **时效：** 7-30 天（平均 12-18 天）
- **适合：** 跨境电商主流选择
- **优点：** 可追踪、时效稳定、价格便宜
- **限制：** 单件≤2kg，长+宽+高≤90cm

### 🚀 最快：DHL/FedEx
- **费用：** ¥180-300 ($25-43)
- **时效：** 3-7 天
- **适合：** 紧急样品、高客单价产品
- **缺点：** 贵，可能产生关税预付费

### 🎯 平衡选择：云途/燕文专线
- **费用：** ¥80-140 ($11-20)
- **时效：** 10-20 天
- **适合：** 跨境电商批量发货
- **优点：** 价格接近邮政，时效更稳定
- **说明：** 需要找货代合作（个人也可发）

### ePacket 费用估算
```

续重 150g：¥75 × 3 = ¥225
合计：¥248 ≈ $34
```

**但！** 实际找货代拿协议价，1kg 以内通常一口价：
- **美国：** ¥80-120 ($11-17)
- **加拿大：** ¥100-140 ($14-20)

## ⚠️ 重要注意事项

### 1. 关税政策（2026 年更新）
**美国：**
- ❌ **de minimis $800 豁免已取消**（2025 年 5 月起）
- 所有中国包裹都要正式报关
- 关税：2-16% + 301 关税 7.5-25%
- 报关费：$125-300/票（货代通常包在运费里）

**加拿大：**
- 免税额度：CAD $20（很低）
- 超过$20 可能收关税 + GST/HST（5-15%）
- 报关费：CAD $10-30

### 2. 体积重计算
快递按**实际重**和**体积重**取大者计费：
```
体积重 (kg) = 长 (cm) × 宽 (cm) × 高 (cm) ÷ 5000
```

**举例：** 20cm × 15cm × 5cm 的盒子
```
体积重 = 20 × 15 × 5 ÷ 5000 = 0.3kg
实际重 0.2kg → 按 0.3kg 计费
```

### 3. 禁寄物品
水晶手串一般没问题，但注意：
- ❌ 不能含液体/粉末（如精油）
- ❌ 不能有电池（电子首饰）
- ✅ 纯水晶/宝石/金属都可以

## 💰 实际成本测算（ Etsy 订单）

假设你在 Etsy 卖一条手链：
- **售价：** $45
- **成本：** ¥50 ($7)
- **包装：** ¥5 ($0.7)
- **运费（ePacket 到美国）：** ¥100 ($14)
- **Etsy 佣金：** $45 × 10% = $4.5
- **收款手续费：** $45 × 3.25% = $1.46

**利润计算：**
```
收入：$45
成本：$7 + $0.7 + $14 + $4.5 + $1.46 = $27.66
利润：$45 - $27.66 = $17.34 (约¥125)
利润率：38.5%
```

**如果包邮：** 运费已计入售价，客户感知更好  
**如果不包邮：** 客户另付$12-15 运费，可能影响转化

## 📍 如何发货（实操步骤）

### 方案 A：直接找中国邮政
1. 去当地邮局国际窗口
2. 填写报关单（CN22/CN23）
3. 选择 ePacket 或 EMS
4. 现场称重付费
5. 拿 tracking number

**优点：** 官方渠道，可靠  
**缺点：** 价格贵（无折扣），排队耗时

### 方案 B：找货代（推荐）
1. 淘宝/1688 搜"国际快递货代"
2. 问价：1kg 到美国/加拿大，ePacket/专线
3. 对比 3-5 家，选价格适中 + 评价好的
4. 寄货到货代仓库（国内包邮）
5. 货代打包发货，给你 tracking number

**优点：** 价格低（协议价），上门取件，包报关  
**缺点：** 需要甄别靠谱货代

### 方案 C：云途/燕文等跨境物流
1. 注册账号（官网或 APP）
2. 在线下单，打印面单
3. 货物寄到集货仓
4. 系统自动追踪

**优点：** 系统化，适合批量  
**缺点：** 有最低充值要求（通常¥500-1000）

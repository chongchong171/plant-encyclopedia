# 🚨 识植物头图快速解决方案

**问题：** 所有入口（首页识别、搜索、发现页）的识植物功能都没有头图

**原因：**
1. 云函数 `getCareGuide` 的图片获取逻辑慢（GBIF + 维基百科）
2. 前端在等云函数返回图片，但云函数可能超时或失败
3. 没有快速保底的图片源

---

## ✅ 立即可用的免费方案（按优先级）

### 方案 1：Unsplash 源图（最快，5 分钟上线）⭐⭐⭐⭐⭐

**API：** `https://source.unsplash.com/400x400/?plant,{植物名}`

**优点：**
- ✅ 完全免费，无需 API Key
- ✅ 图片质量高，加载快（CDN）
- ✅ 按植物类型返回相关图片（不是随机）
- ✅ 5 分钟即可上线

**缺点：**
- ⚠️ 图片不一定是对应植物（但相关）
- ⚠️ 需要网络访问 Unsplash

**实现代码：**
```javascript
// 云函数端调用
const unsplashRes = await fetch(`https://source.unsplash.com/400x400/?plant,${encodeURIComponent(plantName)}`, {
  timeout: 5000,
  redirect: 'manual'
});

// Unsplash source 返回 302 重定向到实际图片
if (unsplashRes.status === 302) {
  return unsplashRes.headers.get('location');
}
```

**示例：**
- `https://source.unsplash.com/400x400/?plant,绿萝` → 绿萝相关图片
- `https://source.unsplash.com/400x400/?plant,玫瑰` → 玫瑰相关图片

### 方案 2：Pexels API（免费，高质量）⭐⭐⭐⭐

**API：** `https://www.pexels.com/api/search/?query={植物名}&per_page=1`

**优点：**
- ✅ 完全免费，图片质量极高
- ✅ 有 API 保证稳定性
- ✅ 按关键词搜索相关图片

**缺点：**
- ⚠️ 需要 API Key（免费申请）
- ⚠️ 需要额外配置

**实现代码：**
```javascript
const pexelsRes = await fetch(`https://www.pexels.com/api/search/?query=${encodeURIComponent(plantName)}&per_page=1`, {
  headers: {
    'Authorization': 'YOUR_PEXELS_API_KEY'
  }
});

const data = await pexelsRes.json();
if (data.photos && data.photos.length > 0) {
  return data.photos[0].src.medium;
}
```

**申请 API Key：** https://www.pexels.com/api/

## 📝 前端优化：不等图片，先显示页面

### 当前问题

前端在等云函数返回图片才显示页面，导致：
- 图片加载慢 → 页面一直 loading
- 图片失败 → 页面空白

### 解决方案

**两步加载：**
1. **先显示页面**（用默认图或占位图）
2. **异步加载真实图片**（不阻塞页面）

```javascript
// 前端代码
async searchPlant(keyword) {
  // 1. 立即显示页面（用默认图）
  this.setData({
    loading: false,
    plant: {
      name: keyword,
      imageUrl: 'https://images.unsplash.com/photo-1596724857861-d277d5e25133?w=400&q=80'
    }
  });
  
  // 2. 异步加载真实图片（不阻塞）
  this.loadRealImage(keyword);
}

async loadRealImage(keyword) {
  try {
    const result = await wx.cloud.callFunction({
      name: 'getCareGuide',
      data: { plantName: keyword }
    });
    
    if (result.result.imageUrl) {
      this.setData({ 'plant.imageUrl': result.result.imageUrl });
    }
  } catch (e) {
    console.log('图片加载失败，保持默认图');
  }
}
```

## 二、云数据库配置（热更新，无需审核）

### 集合 1：`app_config`（业务配置）

**操作路径：** 微信开发者工具 → 云开发 → 数据库 → `app_config`

**文档 ID：** `global`

**默认结构：**

```json
{
  "_id": "global",
  "aiModel": {
    "name": "qwen-turbo",
    "endpoint": "https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation",
    "maxTokens": 1500,
    "temperature": 0.7
  },
  "plantnet": {
    "dailyLimit": 500,
    "confidenceThreshold": 0.3
  },
  "careDefaults": {
    "wateringDays": 7,
    "fertilizingDays": 30,
    "warningDays": 2
  },
  "ui": {
    "homeBannerText": "拍照识别植物，一键获取养护指南",
    "profileSubtitle": "每天认识一种植物 🌿",
    "profileBio": "AI 植物管家 v1.0.0",
    "emptyPlantTip": "去首页识别植物，添加第一盆吧"
  },
  "subscriptionTemplateId": "XEGNQZUcsrWTE9JKZG088lSpQE2jjzR9JF0pAofOPgY"
}
```

**常见修改场景：**

| 场景 | 修改字段 | 说明 |
|-----|---------|------|
| 更换大模型 | `aiModel.name` | 如改为 `qwen-plus`、`gpt-4o` 等 |
| 调整模型参数 | `aiModel.maxTokens` / `aiModel.temperature` | 控制输出长度和创意度 |
| 调整识别额度 | `plantnet.dailyLimit` | 每日免费识别次数上限 |
| 修改首页标语 | `ui.homeBannerText` | 首页宣传文案 |
| 修改个人中心文案 | `ui.profileSubtitle` / `ui.profileBio` | 副标题和版本说明 |
| 更换订阅消息模板 | `subscriptionTemplateId` | 微信后台申请新模板后替换 ID |

**前端读取方式：**
```javascript
const { getConfig } = require('../../config/constants');
const modelName = getConfig('aiModel.name', 'qwen-turbo');
```

### 集合 2：`feature_flags`（功能开关）

**操作路径：** 微信开发者工具 → 云开发 → 数据库 → `feature_flags`

**文档 ID：** `default`

**默认结构：**

```json
{
  "_id": "default",
  "shopEnabled": false,
  "rankingEnabled": false,
  "showAiReminder": true,
  "showCommunityGarden": false,
  "enableDiagnosis": true,
  "enableCareGuide": true,
  "enableWateringReminder": true,
  "enableSubscriptionMsg": true,
  "newDiagnosisLayout": false,
  "enableVoiceInput": false
}
```

**开关说明：**

| 开关名 | 当前状态 | 控制范围 |
|-------|---------|---------|
| `shopEnabled` | false | 植物养护商城入口显示/隐藏 |
| `rankingEnabled` | false | 花友排行榜是否启用真实数据（筹备中页面不受影响） |
| `showAiReminder` | true | AI 智能提醒功能 |
| `showCommunityGarden` | false | 社区花园相关功能 |
| `enableDiagnosis` | true | 植物诊断功能 |
| `enableCareGuide` | true | 养护指南功能 |
| `enableWateringReminder` | true | 浇水提醒功能 |
| `enableSubscriptionMsg` | true | 订阅消息推送 |
| `newDiagnosisLayout` | false | 新诊断页面布局（实验性） |
| `enableVoiceInput` | false | 语音输入功能（实验性） |

**前端读取方式：**
```javascript
const { isFeatureEnabled } = require('../../config/constants');
if (isFeatureEnabled('shopEnabled', false)) {
  // 显示商城入口
}
```

**当前已接入开关的页面：**
- `pages/profile/profile.wxml`：花友排行榜入口受 `showRanking` 控制（v1.0 默认显示筹备中页面）

## 三、云函数环境变量（热更新，无需审核）

**操作路径：** 微信开发者工具 → 云开发 → 云函数 → 选择函数 → 版本与配置 → 环境变量

**当前已使用的环境变量：**

| 变量名 | 用途 | 修改场景 |
|-------|------|---------|
| `QWEN_API_KEY` | 通义千问 API Key | 更换账号/密钥泄露时轮换 |
| `PLANTNET_API_KEY` | PlantNet 识别 API Key | 额度用完或更换账号 |
| `BAIDU_API_KEY` | 百度语音/图像 API Key | 更换百度账号 |
| `BAIDU_SECRET_KEY` | 百度 Secret Key | 同上线 |

**注意：** 环境变量修改后需要**重新部署**对应云函数才生效，但不需要小程序审核。

## 五、上线后操作手册

### 场景 A：更换大模型（无需审核）

1. 打开微信开发者工具 → 云开发 → 数据库
2. 找到 `app_config` 集合 → `global` 文档
3. 修改 `aiModel.name` 为新模型名（如 `qwen-plus`）
4. 如需调整参数，同步修改 `aiModel.maxTokens` / `aiModel.temperature`
5. 保存后立即生效，所有用户下次调用云函数时自动使用新配置

### 场景 B：上线植物商城（需要审核）

1. **开发阶段（本地）：**
   - 开发 `pages/shop/shop.wxml` 真实商城页面
   - 开发对应的云函数（如 `getProducts`、`createOrder`）
   - 在 `feature_flags` 中把 `shopEnabled` 设为 `true`

2. **提审阶段：**
   - 上传代码 → 微信公众平台 → 提交审核
   - 审核通过后发布

3. **上线后如需紧急关闭：**
   - 无需重新审核！直接修改 `feature_flags` 中 `shopEnabled` 为 `false`
   - 入口立即隐藏，已打开页面的用户刷新后也不再显示

### 场景 C：调整首页文案（无需审核）

1. 云开发数据库 → `app_config` → `global`
2. 修改 `ui.homeBannerText` 为新文案
3. 保存后，用户下次启动小程序或切换页面时生效

### 场景 D：调整识别额度（无需审核）

1. 云开发数据库 → `app_config` → `global`
2. 修改 `plantnet.dailyLimit` 为新的数值
3. 保存后立即生效

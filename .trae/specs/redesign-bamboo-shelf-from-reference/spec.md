# 重新设计竹子架 - 基于参考图规格

## Why
当前竹子架的视觉效果不符合用户的期望。用户提供了一张真实的竹编植物架参考图，需要按照该参考图的标准重新设计，将颜色改为青绿色。

## 参考图分析

### 竹子立柱特征：
1. **真实竹子质感**：有明显的**凸起竹节**（环状凸起，不是颜色变化）
2. **粗细变化**：竹节处明显比竹身**粗一圈**
3. **竹节间距**：均匀分布，约3-4个竹节
4. **颜色**（需改为）：青绿色系（#4a7a3a ~ #6b9b5a）
5. **形态**：略微自然的弯曲感

### 藤蔓装饰特征：
1. **缠绕方式**：藤蔓**沿着竹子立柱缠绕向上**
2. **主茎可见**：有清晰的藤蔓主茎，蜿蜒曲折
3. **叶子生长**：叶子从主茎上长出，**不是悬空的**
4. **数量适中**：每侧约4-5片叶子，自然分布
5. **整体感觉**：自然、清新、不夸张

### 置物架结构：
1. **多层设计**：4层置物架
2. **编织/镂空效果**：每层是**竹编纹理**（交叉网格）
3. **横梁**：也是竹子材质，有竹节
4. **顶部**：可选的编织顶棚

## What Changes
- 完全重写竹子立柱样式（模拟真实竹节凸起效果）
- 重新设计藤蔓装饰（缠绕式，主茎+叶子）
- 添加竹编纹理效果到置物架层板
- 颜色统一为青绿色系

## Impact
- Affected code:
  - `pages/my-plants/my-plants.wxss` - 主要样式修改
  - `pages/my-plants/my-plants.wxml` - 结构可能需要调整

## ADDED Requirements

### Requirement: 真实竹子立柱效果
The system SHALL 渲染具有真实竹子质感的立柱

#### Scenario: 竹子外观
- **GIVEN** 用户查看植物架
- **THEN** 立柱显示明显的**凸起竹节**（比竹身宽8-10rpx）
- **AND** 竹节处有立体阴影效果
- **AND** 颜色为青绿色渐变（#3d6b30 → #5a8f4a → #7cb342）
- **AND** 整体看起来像真实竹子，不是塑料棍

### Requirement: 缠绕式藤蔓装饰
The system SHALL 显示沿竹子缠绕的自然藤蔓

#### Scenario: 藤蔓外观
- **GIVEN** 用户查看植物架两侧
- **THEN** 能看到**蜿蜒的藤蔓主茎**（深绿色，12-14rpx宽）
- **AND** 主茎沿着竹子立柱**螺旋上升**
- **AND** 叶子从主茎上长出（每侧4-5片）
- **AND** 叶子与主茎连接紧密，不悬空
- **AND** 整体看起来自然协调

### Requirement: 竹编置物架效果
The system SHALL 显示具有竹编纹理的置物架层板

#### Scenario: 层板外观
- **GIVEN** 用户查看置物架
- **THEN** 层板显示**交叉网格纹理**（模拟竹编）
- **AND** 镂空透明背景
- **AND** 边缘有竹子横梁包裹

## ⚠️ 核心原则（必须遵守）

### 费用红线 ⭐⭐⭐
- **所有费用操作必须提前确认**（云函数调用、API 使用、云存储等）
- **API Key 不硬编码**（全部放在云函数环境变量）
- **使用新 API 前必须确认费用**

### 开发规范 ⭐⭐
- **修改功能前先看旧版本**（不要破坏已有的功能）
- **小步提交代码**（每次只做一个改动，方便回滚）
- **教训必须记录**（每个错误都要写到 lessons/ 目录）

### 用户体验 ⭐⭐
- **统一使用自定义加载动画**（不要用 wx.showLoading）
- **AI 内容显示前清理 markdown 标记**（`**`, `##` 等）
- **搜索和识别是两个独立功能**（不要混为一谈）
